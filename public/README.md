# Public Knowledge Directory

This directory contains shared knowledge base for agents.

## Structure

```
public/
├── common/      # Common knowledge
├── domain/      # Domain-specific knowledge
├── faq/         # Frequently asked questions
├── prompt/      # Prompt templates
├── rag/         # RAG-related knowledge
├── security/    # Security guidelines
├── skills/      # Skill documentation
└── tutorial/    # Tutorials and guides
```

## ⚠️ Important

- **Do not commit** JSON data files to git
- All `*.json` and `*.md` files are ignored
- Only directory structure and `.gitkeep` are tracked

## 中文

此目录包含 Agent 的共享知识库。

**注意：**
- 不要将 JSON 数据文件提交到 git
- 所有 `*.json` 和 `*.md` 文件都被忽略
- 只保留目录结构和 `.gitkeep`
