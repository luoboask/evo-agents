## 📊 安装过程

### 1. 克隆模板
```bash
git clone --depth 1 https://github.com/luoboask/evo-agents.git ~/.openclaw/workspace-demo2-agent
```
**结果：** ✅ 成功

### 2. 创建目录结构
```bash
mkdir -p memory/weekly memory/monthly memory/archive
mkdir -p data/index data/demo2-agent
```
**结果：** ✅ 成功

### 3. 注册 OpenClaw agent
```bash
openclaw agents add demo2-agent --workspace "$(pwd)" --non-interactive
```
**结果：** ✅ 成功

### 4. 测试基本功能
```bash
python3 scripts/session_recorder.py -t event -c 'demo2-agent 初始化完成' --agent demo2-agent
python3 scripts/unified_search.py '初始化' --agent demo2-agent --limit 3
python3 scripts/memory_stats.py --agent demo2-agent
```
**结果：** ✅ 全部通过
