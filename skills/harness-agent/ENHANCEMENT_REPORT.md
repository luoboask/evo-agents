## 🔧 任务 1: 修复领域检测问题 ✅

### 问题描述
原领域检测无法识别"写一个 Hello World 函数"为编程任务。

### 根本原因
关键词映射缺少常见编程相关词汇：
- ❌ 缺少："函数"、"脚本"、"程序"、"算法"等
- ❌ 缺少：编程语言名称（Python、JavaScript 等）
- ❌ 缺少：通用动词（"编写"、"实现"等）

### 解决方案

**增强关键词映射**（从 5 个扩展到 30+ 个关键词）：

```python
'programming': [
    # 核心动词
    '代码', '开发', '编程', '编写', '实现', '构建', '搭建',
    # 名词 - 软件/网站
    '网站', 'APP', '应用', '软件', '系统', '平台',
    # 名词 - 技术组件
    'API', '数据库', '接口', '模块', '功能', '服务',
    # 名词 - 代码单元 ⭐ 新增
    '函数', '脚本', '程序', '算法', '类', '方法',
    # 语言名称 ⭐ 新增
    'Python', 'JavaScript', 'Java', 'Go', 'Rust', 'C++',
]
```

**添加 Layer 2: 任务模式识别**：
```python
patterns = {
    'build_pattern': ['创建', '开发', '构建', '实现', '搭建', '写一个', '做一个'],
    'analyze_pattern': ['分析', '研究', '调查', '评估', '总结', '找出'],
    'review_pattern': ['审查', '检查', '审核', '验收', '评审', '查看'],
    'create_content_pattern': ['撰写', '创作', '设计', '制作', '编写'],
```

## 🎨 任务 2: 创建自定义领域插件 ✅

### 插件信息

| 属性 | 值 |
|------|-----|
| **名称** | self-media-content |
| **中文名称** | 自媒体内容创作 |
| **版本** | 1.0.0 |
| **适用场景** | 公众号/短视频/小红书/播客等内容创作 |

### 核心功能

#### 1️⃣ 任务分解模板（6 步流程）

```
1. 内容定位与受众分析
   - 目标受众画像
   - 内容调性确定
   - 平台特性适配

2. 选题与角度策划
   - 热点话题结合
   - 独特视角切入
   - 价值点提炼

3. 内容大纲设计
   - 标题设计（3-5 个备选）
   - 开头钩子（前 3 秒抓住注意力）
   - 主体结构
   - 结尾引导

4. 内容创作执行
   - 初稿撰写
   - 配图/配视频
   - 金句打磨

5. 优化与发布准备
   - SEO 优化
   - 标签选择
   - 发布时间

6. 数据追踪计划
   - 核心指标
   - A/B 测试
   - 复盘模板
```

#### 2️⃣ 验收标准（9 项）

```
✅ 内容定位清晰
✅ 标题吸引力（至少 3 个备选）
✅ 开头钩子强力（前 3 秒/前 50 字）
✅ 内容价值明确
✅ 结构逻辑清晰
✅ 结尾引导有效
✅ SEO 优化到位
✅ 符合平台规范
✅ 原创性保证（查重率<10%）
```

#### 3️⃣ 领域工具集（7 类）

```
📊 内容灵感工具 - 今日热榜/新榜/灰豚数据
✍️ 标题生成器 - AI 辅助创作
🎨 排版工具 - 秀米/135 编辑器/Canva
🖼️ 配图工具 - Unsplash/Pexels/稿定设计
🎬 视频剪辑 - 剪映/必剪/CapCut
📈 数据分析 - 新榜/蝉妈妈/飞瓜数据
🔍 SEO 优化 - 5118/爱站网
```

#### 4️⃣ 最佳实践（8 条）

```
💡 黄金 3 秒原则 - 开头必须在前 3 秒抓住注意力
💡 情绪价值优先 - 触发用户情感共鸣
💡 干货密度控制 - 每 500 字至少 1 个知识点/金句
💡 视觉化表达 - 多用图表/案例/对比
💡 互动设计 - 在关键节点设置互动点
💡 持续系列化 - 打造 IP 辨识度
💡 数据驱动迭代 - 根据数据反馈优化内容
💡 合规第一 - 严格遵守广告法和平台规则
```

#### 5️⃣ 智能质检方法（5 种）

| 质检项 | 检查逻辑 | 评分标准 |
|--------|---------|----------|
| **标题质量** | 数量 + 数字 + 情绪词 | ≥3 个标题 (30 分) + 有数字 (35 分) + 有情绪 (35 分) |
| **开头钩子** | 类型 + 长度 | 疑问/故事/震惊/数据类，50-100 字最佳 |
| **内容价值** | 价值点 + 案例 + 可操作建议 | 每个价值点 15 分，满分 100 |
| **SEO 优化** | 关键词密度 + 标签数 | 密度 1-3% (50 分) + 3-5 个标签 (50 分) |
| **原创性** | 查重相似度 | <10% 通过，每超 1% 扣 2 分 |

#### 6️⃣ 平台特定配置（4 大平台）

| 平台 | 最佳时间 | 标题长度 | 内容特点 | 关键提示 |
|------|---------|---------|---------|---------|
| **微信公众号** | 20:00-22:00 | ≤64 字 | 深度长文 | 摘要补充标题未表达的信息 |
| **小红书** | 19:00-21:00 | ≤20 字 | 图文笔记 | 标题吸睛用 emoji，首图最重要 |
| **抖音** | 18:00-20:00 | ≤55 字 | 15-60 秒视频 | 前 3 秒必须有钩子 |
| **B 站** | 18:00-20:00 | ≤80 字 | 3-10 分钟视频 | 封面要有冲击力 |
