### 📋 管理命令

```bash
# 查看所有任务
openclaw cron list

# 禁用某个任务
openclaw cron update --jobId <任务 ID> --patch '{"enabled": false}'

# 启用某个任务
openclaw cron update --jobId <任务 ID> --patch '{"enabled": true}'

# 立即运行某个任务
openclaw cron run --jobId <任务 ID>

# 删除某个任务
openclaw cron remove --jobId <任务 ID>
```
