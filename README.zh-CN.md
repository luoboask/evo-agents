## 📚 记忆架构

### 分层记忆系统

```
┌─────────────────────────────────────────────────────────┐
│                    用户查询                              │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│              搜索策略                                    │
│  月 → 周 → 日 → 全量扫描                                 │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│              记忆层级                                    │
├─────────────────────────────────────────────────────────┤
│  月摘要 (2 个月)        → 概览                           │
│  周摘要 (8 周)          → 总结                           │
│  日摘要 (14 天)         → 详情                           │
│  MEMORY.md  (永久)      → 长期记忆                       │
└─────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────┐
│              存储                                        │
├─────────────────────────────────────────────────────────┤
│  memory/monthly/*.md        (月摘要)                    │
│  memory/weekly/*.md         (周摘要)                    │
│  memory/YYYY-MM-DD.md       (每日记忆)                  │
│  data/*/memory/memory_stream.db  (共享记忆)             │
└─────────────────────────────────────────────────────────┘
```

### 记忆压缩流程

```
每日记忆 (memory/YYYY-MM-DD.md)
         ↓ (每天 09:30)
   增量压缩
         ↓
   共享记忆 (memory_hub)
         ↓ (每周日 03:00)
   周摘要 (memory/weekly/)
         ↓ (每月 1 号 04:00)
   月摘要 (memory/monthly/)
         ↓
   自动清理 (14 天/8 周/2 个月)
```

### 压缩时间表

| 类型 | 时间 | 保留 | 自动清理 |
|------|------|------|---------|
| **每日** | 每天 09:30 | 14 天 | ✅ 是 |
| **每周** | 周日 03:00 | 8 周 | ✅ 是 |
| **每月** | 1 号 04:00 | 2 个月 | ✅ 是 |

## ✨ 核心特性

### 1. 多 Agent 架构

- 🔒 **数据隔离** - 每个 Agent 独立工作区和数据库
- 📦 **共享技能** - 所有 Agent 通用能力
- 🛠️ **简单设置** - 一行命令安装

### 2. Harness Agent 插件

**8 个领域插件:**

| 领域 | 用途 | 示例 |
|------|------|------|
| **Programming** 💻 | 软件开发 | `--domain programming` |
| **E-commerce** 🛒 | 商品管理 | `--domain ecommerce` |
| **Data Analysis** 📊 | BI 分析 | `--domain data_analysis` |
| **DevOps** 🔧 | CI/CD 部署 | `--domain devops` |
| **Marketing** 📢 | 营销活动 | `--domain marketing` |
| **Content** ✍️ | 文章脚本 | `--domain content_creation` |
| **Self-Media** 📱 | 自媒体 | `--domain self_media` |

### 3. 高级记忆系统

- 📅 **分层压缩** - 日→周→月
- 🔍 **分层搜索** - 月→周→日→全量
- 🗑️ **自动清理** - 14 天/8 周/2 个月
- 💾 **共享存储** - SQLite memory_hub

### 4. 自进化系统

- 📊 **RAG 评估** - 跟踪检索质量
- 🔧 **自动调优** - 优化参数
- 📈 **持续改进** - 从交互中学习

## 🔧 使用

### 安装

```bash
# 安装
./install.sh my-agent

# 激活功能
./scripts/core/activate-features.sh
```

### 记忆管理

```bash
# 每日压缩
python3 scripts/core/memory_manager.py --daily

# 周压缩
python3 scripts/core/memory_manager.py --weekly

# 月压缩
python3 scripts/core/memory_manager.py --monthly

# 搜索记忆
python3 scripts/core/memory_manager.py --search "关键词"
```

### Cron 任务（自动配置）

```
*/30 * * * *   会话扫描
0 9:30 * * *   每日记忆压缩
0 3 * * 0      周记忆压缩
0 4 1 * *      月记忆压缩
0 23 * * *     夜间进化
```
