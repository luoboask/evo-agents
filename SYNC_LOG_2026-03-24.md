### 2. 新增共享库

**libs/memory_hub/** - 记忆管理共享库

```
libs/memory_hub/
├── __init__.py          # 包入口，导出 MemoryHub 等
├── hub.py               # 核心 MemoryHub 类
├── storage.py           # 存储管理（支持语义搜索）
├── knowledge.py         # 知识管理接口
├── evaluation.py        # RAG 评估接口
└── models.py            # 数据模型定义
```

**功能:**
- 统一的记忆 CRUD 操作
- 关键词搜索 + 语义搜索（Ollama）
- Embedding 缓存机制
- 知识管理接口
- RAG 评估集成

**使用方式:**
```python
from libs.memory_hub import MemoryHub

hub = MemoryHub(agent_name="ai-baby")
hub.add_memory("内容", type="observation")
results = hub.search("查询", top_k=5, semantic=True)
```
